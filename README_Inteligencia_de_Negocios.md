# Actividad: Uso de GitHub y Streamlit

## Inteligencia de Negocios – Universidad Panamericana

### Descripción

Este repositorio corresponde a una actividad de la materia de **Inteligencia de Negocios** de la **Universidad Panamericana**.

El objetivo principal es comprender cómo utilizar **GitHub** para almacenar y compartir proyectos, así como **Streamlit** para crear y visualizar aplicaciones web de manera sencilla.

La actividad se enfoca en practicar el flujo básico de trabajo entre ambas herramientas: crear un repositorio, subir archivos, organizar el proyecto y ejecutar una aplicación en Streamlit.

---

## Objetivo de la actividad

El objetivo de este proyecto es aprender a:

- Crear y administrar un repositorio en GitHub.
- Subir archivos de un proyecto a GitHub.
- Documentar un proyecto mediante un archivo `README.md`.
- Utilizar Streamlit para desarrollar una aplicación web básica.
- Ejecutar una aplicación localmente y comprender su posible despliegue en línea.

---

## Herramientas utilizadas

Para esta actividad se utilizaron las siguientes herramientas:

- **GitHub:** plataforma para guardar, controlar versiones y compartir proyectos.
- **Streamlit:** librería de Python que permite crear aplicaciones web interactivas de forma rápida.
- **Python:** lenguaje de programación utilizado para desarrollar la aplicación.
- **Visual Studio Code / Google Colab / Editor de código:** utilizado para escribir y editar los archivos del proyecto.

---

## Estructura del proyecto

El repositorio puede estar organizado de la siguiente manera:

```text
proyecto-streamlit/
│
├── app.py
├── requirements.txt
└── README.md
```

### Descripción de archivos

- `app.py`: archivo principal de la aplicación desarrollada en Streamlit.
- `requirements.txt`: archivo que contiene las librerías necesarias para ejecutar el proyecto.
- `README.md`: documento explicativo del proyecto, sus objetivos y forma de uso.

---

## Instalación y ejecución

Para ejecutar este proyecto de manera local, se pueden seguir los siguientes pasos:

### 1. Clonar el repositorio

```bash
git clone URL_DEL_REPOSITORIO
```

### 2. Entrar a la carpeta del proyecto

```bash
cd nombre-del-repositorio
```

### 3. Instalar las dependencias

```bash
pip install -r requirements.txt
```

### 4. Ejecutar la aplicación en Streamlit

```bash
streamlit run app.py
```

Después de ejecutar este comando, Streamlit abrirá la aplicación en el navegador.

## Uso de Streamlit

Streamlit permite convertir archivos de Python en aplicaciones web interactivas.

Con esta herramienta se pueden mostrar textos, tablas, gráficas, botones, filtros y otros elementos visuales sin necesidad de desarrollar una página web desde cero.

Un ejemplo básico de código en Streamlit es:

```python
import streamlit as st

st.title("Mi primera aplicación en Streamlit")
st.write("Esta aplicación fue creada como parte de la materia de Inteligencia de Negocios.")
```
